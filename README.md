# SocketChat
Chat application using socket.io and express

Simple chat application for learning socket.io. Deployed on heroku.

Current features:
  - showing nicknames
  - showing who is typing right now
  - unique nicknames
  - showing who entered/left chat

Features to implement:

  - private rooms
